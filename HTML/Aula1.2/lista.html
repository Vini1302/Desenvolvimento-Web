Elemento De lista de descrição

TCP
Transmission Control Protocol é um procolo usado junto com o
protocolo de Intenet (ip) para enviar dados na forma de unidades
de mensagens, chamadas de pacotes, entre computadores pela Internet.
IP
Internet Procotol é um protocolo pelo qual dados são enviados de
um computador para outro na Internet. Cada computador na Internet
é indentificado por um endreço de IP único.
HTTP
Hypertext Transfer Protocol é um protocolo usado para tocas de
arquivos de texto, imagens, gráficas, som, vídeo e ou tipos de
